# Quick Start Guide

## üöÄ Immediate Steps to Get Your Portfolio Live

1. **Replace Placeholder Images**
   - Add your professional headshot as `assets/images/profile.jpg`
   - Add project screenshots to the images folder
   - Update favicon in `assets/images/favicon.ico`

2. **Update Personal Information**
   - Edit your name, title, and description in `index.html`
   - Update contact details (email, phone, location)
   - Replace social media links with your actual profiles

3. **Deploy to GitHub Pages**
   ```bash
   # Initialize git repository
   git init
   
   # Add all files
   git add .
   
   # Commit changes
   git commit -m "Initial portfolio setup"
   
   # Create GitHub repository and push
   git remote add origin https://github.com/behrouz-mousavi/behrouz-mousavi.github.io.git
   git branch -M main
   git push -u origin main
   ```

4. **Enable GitHub Pages**
   - Go to repository Settings ‚Üí Pages
   - Source: Deploy from a branch ‚Üí main ‚Üí / (root)
   - Your site will be live at: `https://yourusername.github.io/`
